# Changelog

모든 주요 변경 사항이 이 문서에 기록됩니다.

## [2.0.0] - 2026-02-23

### 🚀 대규모 업데이트 (Major Overhaul)

- **모듈화 및 빌드 시스템 전환**: 코드베이스를 분리하고 Webpack 기반의 빌드 시스템을 도입하여 유지보수성과 성능을 높였습니다.
- **UI/UX 전면 개편**: List ↔ Grid 뷰 전환, 동적 썸네일 비율 조절 및 메인 툴바 UI 등이 윈도우 탐색기 수준으로 모던하게 개선되었습니다.

### ✨ 새로운 기능 (New Features)

- **자유로운 창 조작**: 창을 8방향으로 자유롭게 크기 조절하거나 창의 위치를 자유롭게 이동 할 수 있는 핸들러가 구현되었습니다.
- **Lasso 다중 선택**: 바탕화면처럼 마우스를 드래그하여 파일과 폴더를 다중 선택할 수 있는 툴 기능이 추가되었습니다.
- **트리 구조 사이드바**: 왼쪽 사이드바에 폴더 트리를 제공하며, **사이드바-파일 리스트 간 폴더 단위 드래그 앤 드롭 이동**이 딜레이 없이 빠르게 가능합니다.
- **고급 필터 옵션**: 파일 종류 확인 창 대신 크기, 날짜, 확장자, 정규식 이름 등의 복합 조건으로 항목을 걸러내는 고급 필터링 기능을 추가했습니다.
- **미디어 특화 내장 플레이어 (Plyr.js)**:
  - 단순한 비디오 태그 대신 아름답고 다기능을 지원하는 퀄리티 높은 HTML5 플레이어 UI를 통합했습니다.
  - 이전/다음 영상, PIP 지원 컨트롤, 반복 무작위 셔플 기능과 같은 미디어 기능이 탑재되었습니다.
  - 외부 플레이어로 보내기 외에 **플레이어 내부의 자막 기능(자막 글씨 크기 조절, 로컬 자막 직접 로드)**을 추가했습니다.
- **다국어 완벽 적용**: 어색한 표현이나 미번역 되었던 전체 UI 요소와 상세 검색 메시지, 툴팁을 4개 국가 (한국어, 영어, 중국어(간체), 일본어) 환경에 맞추어 완벽 번역했습니다.

### 💄 개선 사항 (Improvements)

- 알림창 전면 개체: 일괄 브라우저 기본 경고창(`alert`/`confirm`)을 부드럽고 커스텀 된 모달창 및 토스트 알림으로 교체했습니다.
- 파일명 일괄 변경 미리보기: 파일을 한번에 바꾸기전 이름 변경 결과를 텍스트로 시뮬레이션 해보는 Preview 기능을 추가해 안전장치를 확보했습니다.
- 정렬 로직 및 UI 직관성 개선: 파일 정렬 버튼 위 이름/크기/시간 등 현재 정렬 항목과 화살표 방향(▲/▼)과 확장 리스트가 명확하게 개별 분리되었습니다.
- 중복 파일 정리 고도화: 크기와 업로드 날짜를 기준으로 보존할 원본을 선택하는 정교한 전략 필터 버튼을 신설하여 불필요한 영상을 자동 정리해 줍니다.
- 중국어 외 특수 문자 검색/자동 번역 시 발생하는 구분자(`[[###]]`) 충돌 문제를 개선하여 자막 번역 안정성을 높였습니다.

## [1.2.0] - 2025-11-29

### ✨ 새로운 기능 (New Features)

- **반응형 레이아웃 지원**: 창 크기에 따라 UI가 최적화됩니다. (최소 너비 480px 지원, 모바일 환경 대응)
- **파일 아이콘 확장**: 기존 텍스트 이모지 대신 파일 확장자(영상, 이미지, 오디오, 압축, 문서 등)에 맞는 **SVG 아이콘**을 적용했습니다.
- **커스텀 모달 시스템**: 브라우저 기본 팝업(`alert`, `confirm`, `prompt`)을 디자인된 **전용 모달 창**으로 전면 교체했습니다.
- **Aria2 연결 검증**: 설정 저장 시 Aria2 RPC 서버와의 연결을 테스트하는 기능이 추가되었습니다.
- **보안 강화**: 설정 창의 Aria2 토큰 입력 필드를 비밀번호 타입(`password`)으로 변경하여 노출을 방지했습니다.

### 💄 UI/UX 개선 (Improvements)

- **리스트 뷰 디자인**:
  - 모든 열(파일명, 크기, 길이, 날짜)의 정렬을 **왼쪽 정렬**로 통일했습니다.
  - 데이터 길이에 맞춰 열 너비를 최적화하고 가독성을 높였습니다.
  - 스크롤바 디자인을 얇고 모던하게 변경했습니다.
- **플로팅 버튼 개선**: 외부 이미지 로딩 방식에서 **내장 SVG** 방식으로 변경하여, 네트워크 환경에 관계없이 버튼이 즉시 표시되도록 수정했습니다.
- **설정 창 UI**:
  - 현재 스크립트 이름과 버전(`v1.2.0`)이 제목에 표시됩니다.
  - 닫기 버튼과 텍스트가 겹치지 않도록 레이아웃을 조정했습니다.
- **파일명 표시**: 긴 파일명 표시 시 아이콘이 찌그러지거나 사라지는 현상을 수정했습니다.

### 🐛 버그 수정 (Bug Fixes)

- **정렬 어긋남 해결**: 스크롤바 생성 시 헤더와 리스트의 줄이 맞지 않던 문제를 해결했습니다.
- **영상 길이 정보**: 일부 파일의 재생 시간(Duration)이 표시되지 않던 문제를 해결하기 위해 메타데이터 확인 로직을 보강했습니다.
- **키보드 접근성**: 입력 모달(새 폴더 등)에서 `ESC` 키를 눌러 창을 닫을 수 있도록 수정했습니다.
- **안정성 강화**: `GM_getValue` 등 탬퍼몽키 함수 로드 실패 시 스크립트가 멈추지 않도록 안전 장치를 추가했습니다.
